Here you can find instruction for the project.



Languages used for the application: HTML, CSS, JavaScript, SQLite, Eiffel.
Libraries used: HTTP, wsf, wsf_html, default_standalone, sqlite, base, base_pre.


Firstly, you need to open, compile the project and run the application.

Secondly, open the browser and access the link http://localhost:9090.


For queries' results you need to open DB Browser for SQLite or command prompt: sqlite3 (sqlite precompiled binaries).
